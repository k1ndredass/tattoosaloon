# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
pip install -r requirements.txt
```

## Шаг 2: Применение миграций

**Важно:** Если вы получили ошибку `InconsistentMigrationHistory` или `no such table`, выполните следующие действия:

1. **Остановите сервер Django** (если он запущен) - нажмите Ctrl+C в терминале
2. **Удалите старую базу данных:**
   - Windows: удалите файл `db.sqlite3` вручную или запустите `reset_db.bat`
   - PowerShell: запустите `.\reset_db.ps1`
3. **Примените миграции:**

```bash
python manage.py makemigrations
python manage.py migrate
```

**Или используйте готовые скрипты:**
- Windows: `reset_db.bat`
- PowerShell: `.\reset_db.ps1`

## Шаг 3: Создание суперпользователя

```bash
python manage.py createsuperuser
```

Введите имя пользователя, email и пароль для администратора.

## Шаг 4: Создание тестовых данных (опционально)

```bash
python manage.py create_test_data
```

Эта команда создаст:
- 5 стилей татуировок
- 3 тату-мастера (master1, master2, master3, пароль: password123)
- 5 обычных пользователей (user1-user5, пароль: password123)
- 12 татуировок
- 10 записей на сеансы
- 15 отзывов

**Примечание:** Татуировки будут созданы без изображений. Вы можете добавить изображения через админ-панель или через интерфейс мастера.

## Шаг 5: Запуск сервера

```bash
python manage.py runserver
```

## Шаг 6: Откройте браузер

Перейдите на `http://127.0.0.1:8000/`

## Доступ к админ-панели

Админ-панель Django: `http://127.0.0.1:8000/admin/`

Используйте данные суперпользователя, созданного на шаге 3.

## Важные замечания

1. **Изображения:** Для работы с изображениями убедитесь, что директория `media/` существует и доступна для записи. Django создаст её автоматически при первой загрузке файла.

2. **База данных:** По умолчанию используется SQLite. Для продакшена рекомендуется использовать PostgreSQL.

3. **Статические файлы:** В режиме разработки статические файлы обслуживаются автоматически. Для продакшена выполните `python manage.py collectstatic`.

## Структура ролей

- **Администратор:** Полный доступ ко всем функциям управления
- **Тату-мастер:** Может управлять своими татуировками и просматривать свои записи
- **Пользователь:** Может просматривать каталог, создавать записи, оставлять отзывы
